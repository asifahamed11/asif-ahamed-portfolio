# Asif Ahamed Portfolio

Personal portfolio built with **Next.js 14**, **TypeScript**, **Tailwind CSS**, and **Framer Motion**.

## Tech Stack
- Next.js (App Router)
- React + TypeScript
- Tailwind CSS
- Framer Motion
- Lucide Icons

## Getting Started
```bash
npm install
npm run dev
```
Open `https://asifahamed11.github.io/asif-ahamed-portfolio/`.

## Scripts
```bash
npm run dev    # Start dev server
npm run lint   # Run ESLint
npm run build  # Create production build
npm run start  # Start production server
```

## Deployment Notes
This project is configured for static export:
- `output: "export"`
- `basePath: "/asif-ahamed-portfolio"`

After building, deploy the generated `out/` directory to static hosting (e.g., GitHub Pages).

## Content Customization
Update portfolio data in:
- `src/lib/data.ts`

Main sections are composed in:
- `src/app/page.tsx`
